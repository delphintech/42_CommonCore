This project is about creating a simple wireframe isometric model representation of a 3D landï¿¾scape by linking various points (x, y, z) thanks to line segments (edges).

Run:
	make
	./fdf <file_with_coordinates>

Commands:
	* Zoom: i (in), o (out)
	* Translate: Right, Left, Up, Down
	* Change depth: h (higher), l (lower)
	* Rotate: r
	* Change perspective: p
